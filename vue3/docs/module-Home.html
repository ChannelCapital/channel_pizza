Module: Home

Home component of application. Listing, adding to basket, checkout, and order confirmation are handled within this page.

Data

Name Type Default value Description
shop array - retrieved json data of pizza types to populate page is stored as an array
orders array - retrieved json data of to be delivered pizza orders of current user is stored as an array
items object - pizzas added to basket are stored in this array

Computed

Name Type Description
currentUser current user is returned
total total amount of items in basket is returned
loggedIn checks if any user is logged in

Events

Name Payload Type Description
placeOrder items array of items in cart are placed as order
splitJoin String splits string on "," into an array
addToCart item adds item to cart
removeFromCart item removes item from cart
reloadPage reloads page in order to return from order confirmation to Home page
